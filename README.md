# Dotfiles для Hyprland на Arch Linux

Современное, минималистичное и полностью готовое к использованию графическое окружение на Wayland.

## Что включено

- **Оконный менеджер:** Hyprland
- **Статус-бар:** Waybar
- **Лаунчер:** Wofi
- **Терминал:** Kitty
- **Уведомления:** Dunst
- **Блокировка экрана:** Swaylock-effects
- **Управление обоями:** Hyprpaper
- **Скриншоты:** Grim + Slurp + Swappy
- **Оформление:** Arc-Dark тема, иконки Papirus, курсор Bibata
- **Оболочка:** Zsh с автодополнением и подсветкой синтаксиса

## Быстрый старт

1. Установите базовую Arch Linux, создайте пользователя, настройте сеть.
2. Клонируйте этот репозиторий:
   ```bash
   git clone https://github.com/TKakashi/dotfiles.git ~/dotfiles
   cd ~/dotfiles
3. Запустите установочный скрипт:
   ```bash
   chmod +x install.sh
   ./install.sh
4. После завершения перезайдите в систему и выполните Hyprland (или выберите Hyprland в менеджере входа).
